# Projeto Databricks – Consultas de Dados com PySpark

Projeto acadêmico desenvolvido no Databricks utilizando PySpark para consulta e manipulação de dados.

## Tecnologias utilizadas
- Python
- PySpark
- Apache Spark
- Databricks
- Gráficos
  
## Objetivo
Realizar consultas e análises de dados em um ambiente Big Data utilizando PySpark.

## Estrutura do projeto
- `consulta de dados PySpark.ipynb` → Notebook contendo consultas e transformações de dados

## Observações
Projeto exportado do Databricks e versionado no GitHub para fins de portfólio acadêmico e profissional.
